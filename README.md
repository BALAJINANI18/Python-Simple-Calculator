# 🧮 Python Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations using user input and menu-driven choices.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by Zero Handling
- Continuous Execution using While Loop
- Exit Option

## 📚 Concepts Used

- Variables
- User Input (`input()`)
- Type Casting (`int()`)
- While Loop
- If-Elif-Else Statements
- Arithmetic Operators
- Basic Error Handling

## 🛠️ How It Works

1. Enter the first number.
2. Enter the second number.
3. Choose an operation:
   - 1 → Addition
   - 2 → Subtraction
   - 3 → Multiplication
   - 4 → Division
   - 5 → Exit
4. The program displays the result.
5. The calculator continues running until the user chooses Exit.

## 💻 Sample Output

```text
Enter Number 1: 10
Enter Number 2: 5

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit

Enter choice: 1

Result: 15
```
